# 의사결정 나무 실습

인공지능 개론 수업에서 사용하는 노코딩 ID3 의사결정나무 실습 사이트입니다.

## 사이트 구성

1. 해킹메일 데이터로 엔트로피와 정보이득을 계산해 ID3 트리를 완성합니다.
2. 해킹메일 산점도에서 분할선을 직접 움직이며 질문과 분할을 체험합니다.
3. 의사결정나무 핵심 개념을 정리하고 확인 퀴즈를 풉니다.

정보이득 계산 실습: https://xognsdl1019.github.io/information-gain-tree-lab/

산점도 분할 체험: https://xognsdl1019.github.io/information-gain-tree-lab/scatter/

핵심 정리 및 퀴즈: https://xognsdl1019.github.io/information-gain-tree-lab/summary/

## 배포용 빌드

```bash
npm run build
```

모든 데이터와 판정 규칙은 교육을 위해 구성한 가상 사례입니다.
